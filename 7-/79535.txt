die hilfe der software VTPlus für win/tv vers.2.62 16bit
(32bit laeuft instabil bei mir irgendwie instabil)
zum befehl get ist fehlerhaft bzw hat eine option vergessen
der befehl RGUZ xyz erzwingt nicht wie erhofft den senderwechsel,
(jedenfalls bei mir nicht)
so das man dann nicht entsprechend von mehreren sendern die
dazugehörigen videotextseiten zusammen aus einem script heraus abrufen
könnte!
hierzu verwendet man einfach get wxyz303, 
wobei die option wxyz für den gewünschten sender steht
passende senderkennung vorausgesetzt

als eine anregung für nicht boersianer
folgendes beispiel:
spart die fernsehzeitung für den abend ein

VTPLUS SCRIPT
SET MULTICHANNEL YES
GET pro7303
get sat1303

GET FEU444
GET PREM303 
GET VH-1303
