Wie schon im anderen Posting von mir gesagt, es geht um die Stückzahl,
die bei einem echten Börsen-Einzelumsatz über den Tisch geht. Im
BrokerPoker ist man natürlich verleitet, möglicherweise alles auf eine
Karte zu setzten, um dann mit dabei zu sein, wenn man einen Treffer
landet. Wenn man nun aber gleich ein Order für xx Stück im Poker
eingibt, wird dies eben nur dann ausgeführt, wenn in echt tatsächlich
ein Einzelumsatz mit xx Stk oder mehr stattfindet.

Wenn man tatsächlich einen großen Anteil seines Pokerdepots (oder
sogar alles) auf eine Aktie / einen Warrant setzen will, sollte man
entweder sein Order über mehrere Börsenplätze streuen oder mehrere
kleinere Pakete nacheinander ordern. Bei letzterer Alternative kann
man aber die neue Order erst eingeben, wenn die erste ausgeführt
wurde, wenn man mehrere Order auf eine Aktie an einem Börsenplatz
eingibt, wird nur eins ausgeführt (so zumindest meine Erfahrung).
