Hat man genug Geld ist steuerfreies Spekulieren kein 
Problem. Man muss die Spekulationsgeschaefte ueber eine
Firma C in einer Steueroase laufen lassen und diese
Spekulationsfirma muss Tochter einer Firma B mit echter 
Geschaeftstaetigkeit in einem EU-Land sein (also keine
reine Briefkasten-Zwischenfirma), das kein solch strenges 
Aussensteuerrecht wie Deutschland hat. 
Der Besitzer A kann dann praktisch ueberall wohnen, ohne
dass das Einkommen der Firma C besteuert wird (solange es
in C angesammelt und nicht als Dividende ausgeschuettet 
wird).

Fuer Deutsche wichtig ist dass sie keinesfalls die
geschaeftlich aktive Zwischenfirma B weglassen.
Wuerde der deutsche A direkt die Firma C in der 
Steueroase besitzen oder ueber eine nicht aktive
Zwischenfirma, so wuerde ihm das Einkommen von C
steuerlich direkt zugerechnet.
Erst durch Einschaltung von B greift das Steuerrecht
des Landes I, und das sieht Nichtbesteuerung der Tochter
C vor und.
