DataDesign HBCI Transaction Server

Die Firma DataDesign bringt im 02. 11. 20 einen MCR Transaction
Server auf den Markt, der ein Höchstmaß an Sicherheit im Bereich
Homebanking ermöglicht und auf einer modernen flexiblen Architektur
aufsetzen wird. Das Produkt wird als Cluster-Version unter Udilap
verfügbar sein und eine offene flexible Schnittstelle zur Anbindung an
die Host-Architektur der Banken beinhalten.
