Am besten ein Beispiel: 
Angenommen von einem boersennotierten Wertpapier existieren 10.000 Aktien, 
die aktuell mit einem Wert von je 300.00 DM an der Boerse notiert sind. 

Nun kommt ein Geschaeft zustanden, bei dem 2.000 Aktien zum Preis von 
je 290.00 DM den Besitzer wechseln. Wie wuerde anschliessend die neue 
Boersennotierung der Aktie lauten? 

   290.00 DM ?        ( waere ein ziemlicher Kurssprung ) 

   Oder 298.00 DM ?   ( = [ 8.000 x 300 DM + 2.000 x 290 DM ] / 10.000 ) 

   Oder xyz DM ?      ( eine ganz andere Formel; gleitend womoeglich? ) 

Und: Kann es vorkommen, dass sich eine Boersennotierung aendert, _ohne_ 
dass ein Handelsgeschaeft zustanden kam, dass also schon das blosse 
Veroeffentlichen von Kauf- oder Verkaufsangeboten Einfluss auf die 
Boersennotierung hat (_ohne_ dass ein Besitzwechsel stattgefunden hat) ?
