Ob Windows 98 einen ähnlichen Run erleben wird wie seinerzeit Windows
95, wage ich zu bezweifeln. Ich selbst benutze jetzt Windows 98 seit
einem halben Jahr. Das Betriebssystem läuft wirklich erstaunlich
stabil und unterstützt auch die neueste Hardware, doch hat sich im
Vergleich zum Sprung von Windows 3.11 auf Windows 95 diesmal wenig
revolutionär verändert. Wer also z.B. bereits den IE4.0 benutzt, der
kann auf Windows 95 verzichten. Natürlich wird Windows 98 ab Sommer
auf den meisten Neu-PCs neu installiert sein und Microsoft
dementsprechend verdienen. Doch hat Microsoft selbst schon
angekündigt, daß Windows 98 schon bald in dem neuen Windows NT
verschmelzen wird. Insgesamt wird bei Microsoft durch Windows 98
allein kein Wachstumsschub zu erwarten sein.
